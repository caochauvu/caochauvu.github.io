Genetics Distances

Counting polymorphic sites (Hamming distance) doesn't work because we don't observe substitutions.

We can observe changes, but not the process that resulted in that change. For all the below example, assumes one ancestor and two descendants that might have a changed base or not.

  • Single substitution: 1 change (A \(\rightarrow\) C), 1 difference.
    • Ancestor: A
    • Descendants: A, C
  • Multiple substitution: 2 changes (A \(\rightarrow\) C \(\rightarrow\) T) 1 difference.
    • Ancestor: A
    • Descendants: A, T
    • Keep in mind that there could be more than 2 changes, we can't observe them without other data.
  • Coincidental substitution: 2 changes (A \(\rightarrow\) G, A \(\rightarrow\) C), 1 difference.
    • Ancestor: A
    • Descendants: G, C
  • Parallel substitution: 2 changes (A \(\rightarrow\) C, A \(\rightarrow\) C), no difference.
    • Ancestor: A
    • Descendants: C, C
  • Convergent substitution: 3 changes (A \(\rightarrow\) C \(\rightarrow\) T, A \(\rightarrow\) T), no difference.
    • Ancestor: A
    • Descendants: T, T
  • Back substitution: 2 changes (A \(\rightarrow\) C \(\rightarrow\) A), no difference.
    • Ancestor: A
    • Descendants: A, A

So let's first define what we want.

  • An evolutionary/ genetic distance based on expected numbers of substitution between the two sequences of interest.
  • This definition depends on a model for how substitutions occur, most likely a Markov one.
  • Let's make the assumption that all mutations happen at an equal rate.

Jukes-Cantor Models:

A G C T
A \(-3\alpha\) \(\alpha\) \(\alpha\) \(\alpha\)
G \(\alpha\) \(-3\alpha\) \(\alpha\) \(\alpha\)
C \(\alpha\) \(\alpha\) \(-3\alpha\) \(\alpha\)
T \(\alpha\) \(\alpha\) \(\alpha\) \(-3\alpha\)

The rows must sum up to zero by convention.

Matrix defines model. Expected # of substitution is:

$$d = \frac{1}{4} \cdot 3(\alpha) \cdot 2t \cdot 4$$
  • \(d\) is genetic distance.
  • \(\frac{1}{4}\) is initial base frequency (assuming equal base frequency)
  • \(3(\alpha)\) is summing over ending state.
  • \(2t\) is total amount of time (between ancestor and descendants)
  • \(4\) is summing over initial state.

Simplifying the formula we get:

$$d=6 \alpha t$$

However we don't know what alpha and t is yet, we need to extract from the data we have so far.

We know how to compute the probability that a given site is polymorphic vs not polymorphic:

$$P_{xy}(\alpha,t)=\frac{3}{4}(1-e^{-8 \alpha t})$$

What if we wanted it of changing from from one base to itself, you divide by 3:

$$P_{xy}(\alpha,t)=\frac{1}{4}(1-e^{-8 \alpha t})$$

Let's go back to any sites though. Suppose we have \(n\) sites. What is the expected # of polymorphic sites under the Jukes-Cantor model?

$$n*P_{xy}(\alpha,t) = n*(3/4)(1-e^{-8 \alpha t})$$

What is the frequency of polymorphic site?

We would just cancel n if we wanted frequency, so it's

$$P_{xy}(\alpha,t)=\frac{3}{4}(1-e^{-8 \alpha t})$$

Let's match observed (D) and theoretical expectations of this frequency:

$$D = \frac{3}{4}(1-e^{-8 \alpha t})$$

We need to solve for \((\alpha)t\):

$$\hat{\alpha(t)} \sim \frac{1}{8}\ln(1-\frac{4}{3}D)$$

This will let us figure out what the product (alpha)t is. It won't let us figure out the individual values, but the product is all we need to calculate.

Plugging the \((\alpha)t\) back into our formula for d:

$$d = 6 \alpha t \Rightarrow \hat{d} = 6 \hat{\alpha(t)} = \frac{6}{8}\ln(1-\frac{4}{3}D)$$

Simplifying into:

$$\hat{d} = \frac{3}{4}\ln(1-\frac{4}{3}D)$$

This equation is also known as the Jukes-Cantor distance (correction).

There however is an issue in that our expected difference and observed difference will diverge, with the expected difference continuing upward while the observed difference will flattened out. The difference between the two is the "correction".

There are two difference ways to vary the model, then one that combines both and one that takes it further.

  • Kimura 2 Parameter (K2P)
    • Allows for transition/ transversion bias.
    • Maybe A prefers to turn into G more than it does C/T. That is \((\alpha) > (\beta)\).
    • The diagonal would not necessarily be equal to \(-3(\alpha)\), it would instead be equal to \(-(\alpha) + 2(\beta)\).
  • Felsenstein(P81)
    • Allow for unequal base frequency.
    • AGCT can have different frequencies.
  • Hasegawa et al. (HKY85)
    • Allow for both unequal base frequencies and different substitution rates for transversions and transitions.
  • General time reversible model (GTR)
    • Allow for all six pairs of substitutions to have different rates in addition to the HKY85.
  • Each of the above models has an evolutionary distance associated with that model: \(d_{JC}\), \(d_{K2P}\), \(d_{F81}\), \(d_{F84}\), \(d_{HKY85}\), \(d_{GTR}\), etc.
  • F84 is the best because it has similar results to HKY85 and GTR but are easier to compute.
